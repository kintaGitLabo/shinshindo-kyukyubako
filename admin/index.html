⚙️ 設定（GitHub保存トークン）

GitHubに保存機能を使うには、Fine-grained PATが必要です。
作成手順: こちらから発行 → Resource owner: shinshindo-okinawa → Only select repositories: shinshindo-kyukyubako → Repository permissions: Contents = Read and write → Generate → 生成された github_pat_... をコピーして下のフォームに貼り付け。
